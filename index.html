Say hello to MetaPipe

The worlds first cloud based VFX and Animation Studio as a Service.

Next

MetaPipe provides easy to deploy, fully virtualized VFX and animation studio infrastructure.

What does that mean to you?

Our easy to use single pane of glass web dashboard manages everything: Users, Groups, permissions, workstations, servers, storage, deployed software...EVERYTHING. Provision high-performance, workstation-class, cloud-based desktops with ease. Network and storage infrastructure is automatically set up and you only pay for the storage space you actually use. Set up new Users and Groups in seconds. Deploy an entire high-end VFX or animation studio in literally minutes from the comfort of your home. Pay-as-you-go for Compute AND Software licensing*. Pay only for the time you actually use. Clear, per-hour pricing simplifies project budgeting.

Provision exactly the size workstations, number of render servers and software stack you need to control costs. Turn-key pipeline templates for common studio types (VFX, Animation, etc.) are included. Basic Version Control of production assets is included. Automatic back up and geo-redundancy of all data is included. An integrated render farm with auto scaling and pay-as-you-go software licensing is included. Even non-technical users can provision complex infrastructure. Setting up new workstations and users from our Administrator Console is as easy as setting up a webmail account.

*From participating software partners. New partners are being added as the deals are negotiated.

The world is your playground.

Securely collaborate with geographically dispersed teams.

Because all infrastructure is in the cloud, including the workstations, there is no need to distribute data to users. There is no need to move data across sites or out of your trusted cloud environment. Users connect to the cloud and their desktops are streamed to them via PCoIP, the highest quality remote protocol available, which includes a build-to-lossless feature for true pixel perfect displays even under challenging network conditions. Users may use either a software only viewer on a traditional computer or a dedicated PCoIP Zero Client to view their cloud desktop.

Turn-key scafold pipeline makes getting started on projects much easier. The fact it's based on Linux makes it a great training ground for students interested in learning high-end pipeline methodologies also.

Let us be your "IT Guys"

IT doesn't need to be rocket science.

MetaPipe explodes the traditional model of IT infrastructure for VFX and animation.

By freeing VFX and animation studios nearly completely from the burdon of IT, MetaPipe makes onboarding new users simple. No need for a dedicated IT team. A TD or Supervisor can set up new workstations and users themselves. All you need is fast internet connection for your physical location and a Zero Client or basic computer capable of running the software client for each user you want to add.

For studios that require integration with their on-site pipeline we offer VPN support, bring-your-own licenses option and integration consulting to get your environment ported to our cloud.
